# euler-haskell

[Beeminder goal][0] | [Problems][1]

## Dev

- Install Haskell using [GHCup][3]
- `ghci ./problems/1/solve.hs` to open the file
- `:r` to reload the file
- `:t` to see the type of anything
- `:i` to see info about type classes

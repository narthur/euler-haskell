# Sums of totients of powers

<p>Let $\varphi(n)$ be Euler's totient function.</p><p>
Let $f(n)=(\sum_{i=1}^{n}\varphi(n^i)) \text{ mod } (n+1)$.</p><p>
Let $g(n)=\sum_{i=1}^{n} f(i)$.</p><p>
$g(100)=2007$.
</p>
<p>
Find $g(5 \times 10^8)$.
</p>
